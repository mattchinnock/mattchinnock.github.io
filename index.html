Critical Thinking

I'm Matt Chinnock, a British born Project Lead and Senior Developer based out of Honolulu, Hawaii. I've helped shape the future of the ice industry, redfinded lead generation in Higher Education, and co-founded an integration platform that connects Salesforce.com natively to other systems. I've worked in factories, offices, accelerators and incubators across the country including the tech capital of the world, San Francisco.

My primary objective is to work on creative projects that inspire, innovate and improve the lives of others.

Creative problem solving

Finding solutions through communication and collobaration. Architecting systems and processes to improve efficiency. Solving problems in new ways by combining my unique experiences and innovative thought processes.

User driven design

Development projects start with User personas, use-cases, User stories and by simplifying specifications into digestible tasks. Meaningful structure behind why we build things keeps projects on time and true to their business requirements.

Agile development

Using Agile tools such as Jira to seemlessly move between story creation, issue tracking, prioritization, development and testing cycles. Taking advantage of deployment tools and version control for a stable, reliable codebase.

Portfolio

I work primarily in the Lead Developer or Project Manager roles, overseeing technical decisions with input from architects, UX designers, graphics designers and the Product Owner. Interested in working with me? Contact me direcly via email.

    Pythio - Customizable Telemetry Solutions

    Pythio specializes in bespoke hardware monitoring, allowing companies to remotely manage and diagnose their hardware around the world.

    Project Description

    The platform manages thousands of devices, each with multiple sensors sending data back to a centralized server through WiFi or the GSM network. Data is stored in a MongoDB database, fed to an EXT JS frontend over websockets and parsed by visualization libraries such as highcharts.js. Pythio allows it's customers to see data about their machines through a web interface as if they were standing in front of it, from anywhere in the world.

    Role

    Full stack developer, UI designer. Building Javascript components to display and modify device data in real time via a custom Java application server. These components have to not only talk with the noSQL database, but also issue commands directly to the physical devices.

    Valence - Salesforce Integration

    Valence is an integration framework built natively in Salesforce. It radically reduces the time needed to build integrations between Salesforce and other systems, and empowers Admins when it comes to data synchronization.

    Project Description

    A large part of my services work at KapuHonu invloved connecting systems together. Valence is an incarnation of the experience gained from building these custom intregrations over and over. It provides an extensive and extensible platform to help admins, developers and execs connect their Salesforce orgs to any REST based system without having to spend the time, money and effort that comes with a completely bespoke solution.

    Role

    As co-founder, my role in the company was to steer user-focused and technical decision making, build partner relationships and help bring the product to market. As head of UI design, I'd craft user experience. As lead frontend developer, building responsive, functional Lightning components.

    Lead Marketplace*

    "Lead Marketplace"* is a Salesforce based app that collects, sanitises and standardises Leads (information about prospective customers). This data is initally gathered from multiple sources in varying formats, filtered, cleaned up and offered to the end user as a servicable group.

    Project Description

    Built using the robust Data Model of Salesforce, the marketplace utilises external integrations with systems such as Heroku and Elesticsearch for large scale data computation that standarises the data. Once Leads have entered the system, the end user has a myriad of tools available to view, edit and manage the Leads. The Lead aquisition process is recorded at every step, offering performance analysis and process efficiency improvements at every interaction.

    Role

    As a developer on this project I was one of two responsible for the UI design. On top of that, writing Visualforce and Apex code providing the frontend with real-time updates to Lead information through Dashboards, charts and cards. I also implemented a CTI (telephony), allowing users to call Leads directly from their record in the application through a custom built Softphone.

    Akoona - Remote Ice Vending

    With one moving part, Akoona ice machines produce an unlimited supply of fresh filtered ice and water while using only 15 percent of the energy used by traditional methods. The web-based monitoring system tracks power, water, plastic bag supply and transactions, and alerts the operator remotely to any issues with production, quality control or servicing.

    Project Description

    The Akoona platform was designed to remotely manage ice machines located around the world. Using a php based frontend, technicians could diagnose hardware problems before they took the trip out to the machine, allowing for more efficient repairs and allowing them to take care of issues before they occurred.

    Role

    As a developer on this project I managed onboarding of new device types, allowing the user to issue commands and see the status of machines through the php based interface.

    Shopify - Ecommerce Websites

    Shopify is an ecommerce platform that allows users to setup, customise and manage their online stores, providing a base of functionality around products, users, shopping carts and more.

    Projects

    Shopify projects in my Portfolio vary depending on the needs of client, but some of my work includes custom theme development in Liquid (Shopify's frontend language), product management, payment gateway configuration etc. I've also lead a number of integration projects, connecting Shopify to other POS systems and extending the functionality of the platform.

    Role

    Primarily Lead Developer, managing client interactions and a team of developers. Using Jira to create stories and lead the team in an Agile environment.

    Example Shopify projects available on request.
